语法错误 2025-05-03

列表索引错误

错误代码：

my_list = [1, 2, 3]
print(my_list[3])

错误原因：列表索引从0开始，最大索引为长度减1。此列表长度为3，有效索引为0、1、2，访问索引3导致IndexError。

逻辑错误 2025-05-07

循环条件错误

错误代码：

i = 0
while i < 10:
    print(i)
    # 缺少i += 1导致无限循环

错误原因：while循环缺少迭代语句，导致变量i始终为0，条件永远成立，程序陷入无限循环。

运行时错误 2025-05-12

除零错误

错误代码：

def divide(a, b):
    return a / b

print(divide(10, 0))

错误原因：在数学运算中除以零是不允许的，会引发ZeroDivisionError异常。应在除法运算前检查除数是否为零。

语法错误 2025-05-15

缩进错误

错误代码：

if True:
print("Hello, world!")
# 缺少缩进，Python要求代码块必须缩进一致

错误原因：Python使用缩进来表示代码块，if语句下的代码必须有一致的缩进。此例中print语句没有缩进，导致SyntaxError。
